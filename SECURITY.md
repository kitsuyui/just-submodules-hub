# Security Policy

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

To report a security issue privately, use [GitHub's private vulnerability reporting](https://github.com/kitsuyui/just-submodules-hub/security/advisories/new).

Include as much of the following information as possible:
- Type of issue (e.g. command injection, path traversal)
- Affected component and version
- Steps to reproduce
- Potential impact

You will receive a response within 7 days. We aim to release a fix within 30 days of confirmation.

## Supported Versions

Only the latest release on `main` is actively supported with security fixes.
